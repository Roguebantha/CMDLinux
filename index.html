Welcome to the LinuxCMD homepage!

This website is for the LinuxCMD game, created by and for the United States Naval Academy, but available here for free.

What is it?

LinuxCMD is a game written in Java to give people a thorough understanding of how a Linux command-line works and how useful it can be, along with teaching 29 different commands, as well as several concepts such as pipes, command history etc.

Sounds good, where do I get it?

LinuxCMD is available straight from this site, as well as being available at the github repos.

Who made this thing again?

This game was created solely by Seth Jenkins (@Roguebantha), but was advised by at the time of writing Lieutenant-Commander-Select Dane Brown at the United States Naval Academy.
